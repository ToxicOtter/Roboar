Sobre o projeto

Protótipo

Desenvolvemos o protótipo de um robô embuído com diferentes sensores para coletar os dados do ambiente. Para deslocar-se, ele conta com 4 rodas acopladas em sua base.

Back-end

Progamamos o servidor em Python. Ele hospeda o site por meio da biblioteca Flask e gerencia o recebimento de dados do protótipo, salvando-os no banco de dados SQLite.

Front-end

Desenvolvemos o site em HTML, estilizando com CSS. Para torná-lo responsivo, usamos a biblioteca Bootstrap do JavaScript, também usada para extruturação.
